About Career Guidance System

The Career Guidance System is a full-stack web application designed to help students make informed decisions about their career paths. Choosing the right career is one of the most important decisions in a student’s life, and this system aims to simplify that process.

This platform provides detailed information about various career fields such as Technology, Medical, Commerce, and Arts. Each career section explains the required education path, essential skills, top institutions, job roles, and expected salary ranges.

The system includes secure user authentication through login and signup functionality, allowing users to access personalized guidance. The application is developed using modern web technologies including HTML, CSS, JavaScript for the frontend, Node.js and Express for the backend, and MongoDB for database management.

This project is developed as a final-year academic project and follows industry-standard practices to ensure scalability, security, and usability.
